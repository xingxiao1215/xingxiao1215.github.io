## 字体设置
```
示例代码：
*倾斜*
```
### 效果展示：*倾斜*
```
示例代码：
_倾斜_
```
### 效果展示：_倾斜_
```
示例代码：
**加粗**
```
### 效果展示：**加粗**
```
示例代码：
***倾斜加粗***
```
### 效果展示：***倾斜加粗***
```
示例代码：
~~下划线~~
```
### 效果展示：~~下划线~~

## 分级标题
```
#      一级标题
##     二级标题
###    三级标题
####   四级标题
#####  五级标题
###### 六级标题
```
### 效果展示：
#      一级标题
##     二级标题
###    三级标题
####   四级标题
#####  五级标题
###### 六级标题

+ 无序列表

- 无序列表

* 无序列表

### 混合使用
```
* 1. 混合使用
* 2. 混合使用
- 1. 混合使用
- 2. 混合使用
+ 1. 混合使用 
  2. 混合使用
+ 1. 混合使用
```
### 效果展示
* 1. 混合使用
* 2. 混合使用
- 1. 混合使用
- 2. 混合使用
+ 1. 混合使用 
  2. 混合使用
+ 1. 混合使用
